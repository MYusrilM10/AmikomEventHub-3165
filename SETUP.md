# Panduan Setup Projek AmikomEventHub

Dokumen ini menjelaskan cara clone, install, dan menjalankan projek Laravel 12 (AmikomEventHub) dari repository Git.

## 📋 Prasyarat

Pastikan tools berikut sudah ter-install di komputer Anda:

| Tool          | Minimum        | Download                                                                  |
| ------------- | -------------- | ------------------------------------------------------------------------- |
| PHP           | 8.2+           | [XAMPP](https://www.apachefriends.org/) / [Laragon](https://laragon.org/) |
| Composer      | 2.x            | [getcomposer.org](https://getcomposer.org/Composer-Setup.exe)             |
| MySQL/MariaDB | 8.0+           | Sudah include di XAMPP/Laragon                                            |
| Node.js & NPM | 18+ (opsional) | [nodejs.org](https://nodejs.org/)                                         |
| Git           | Latest         | [git-scm.com](https://git-scm.com/)                                       |

Verifikasi instalasi:

```bash
php --version        # Harus 8.2+
composer --version   # Harus 2.x
mysql --version      # Harus 8.0+
```

## 🚀 Langkah Setup

### 1. Clone Repository

```bash
git clone <URL_REPOSITORY>
cd 3165_P3/laravel-app
```

### 2. Install Dependencies

```bash
composer install
# Opsional, kalau ada Vite/JS build
npm install
```

### 3. Setup File `.env`

```bash
# Copy file .env.example (kalau ada), atau buat dari scratch
cp .env.example .env
# Windows PowerShell:
# Copy-Item .env.example .env
```

Generate APP_KEY:

```bash
php artisan key:generate
```

### 4. Konfigurasi `.env`

Edit file `.env` dan sesuaikan bagian berikut:

```env
# === APP ===
APP_NAME=AmikomEventHub
APP_ENV=local
APP_KEY=                          # (auto-generated by key:generate)
APP_DEBUG=true
APP_URL=http://127.0.0.1:8000     # Penting! Sesuaikan dengan port Laravel dev server

# === DATABASE (XAMPP) ===
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=eventtiket_db          # Sesuaikan nama DB
DB_USERNAME=root
DB_PASSWORD=                       # Default XAMPP kosong

# === MAIL (Mailtrap Sandbox untuk testing) ===
MAIL_MAILER=smtp
MAIL_HOST=sandbox.smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=your_mailtrap_username
MAIL_PASSWORD=your_mailtrap_password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS="hello@amikomeventhub.com"
MAIL_FROM_NAME="${APP_NAME}"

# === GOOGLE OAUTH (Opsional, untuk SSO) ===
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
GOOGLE_REDIRECT_URI=http://127.0.0.1:8000/auth/google/callback

# === MIDTRANS (Opsional, untuk payment) ===
MIDTRANS_SERVER_KEY=your_midtrans_server_key
MIDTRANS_CLIENT_KEY=your_midtrans_client_key
MIDTRANS_IS_PRODUCTION=false
MIDTRANS_MERCHANT_ID=your_merchant_id

# === QUEUE (untuk background jobs) ===
QUEUE_CONNECTION=database          # atau 'sync' untuk development
```

### 5. Setup Database

Buat database baru di MySQL (via phpMyAdmin atau CLI):

```sql
CREATE DATABASE eventtiket_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

Jalankan migration:

```bash
php artisan migrate
# Kalau ada seeder:
php artisan db:seed
```

### 6. Setup Storage Link

```bash
php artisan storage:link
```

### 7. Clear Cache & Optimize

```bash
php artisan config:clear
php artisan cache:clear
php artisan view:clear
php artisan route:clear
php artisan optimize
```

## ▶️ Menjalankan Projek

### Cara 1: Laravel Dev Server (Recommended)

```bash
php artisan serve
# Akses di http://127.0.0.1:8000
```

### Cara 2: Apache XAMPP dengan Reverse Proxy (Opsional)

Kalau Anda ingin akses tanpa port (misal `http://localhost/`), setup reverse proxy di Apache:

1. Edit `C:\xampp\apache\conf\httpd.conf`:
    - Uncomment: `LoadModule proxy_http_module modules/mod_proxy_http.so`
    - Uncomment: `LoadModule proxy_http2_module modules/mod_proxy_http2.so`

2. Edit `C:\xampp\apache\conf\extra\httpd-vhosts.conf`, tambahkan di akhir:

    ```apache
    <VirtualHost *:80>
        ServerName localhost
        ServerAlias 127.0.0.1
        ProxyPreserveHost On
        ProxyPass /phpmyadmin !
        ProxyPass /licenses !
        ProxyPass /webalizer !
        ProxyPass / http://127.0.0.1:8000/
        ProxyPassReverse / http://127.0.0.1:8000/
        ErrorLog "logs/laravel-reverse-error.log"
        CustomLog "logs/laravel-reverse-access.log" common
    </VirtualHost>
    ```

3. Restart Apache.

## 🔧 Troubleshooting

### "Route not defined" / 500 Error

```bash
php artisan route:clear
php artisan config:clear
php artisan view:clear
```

### Storage/Gambar Tidak Muncul

```bash
php artisan storage:link
```

### Permission Error (Linux/Mac)

```bash
chmod -R 775 storage bootstrap cache
chown -R $USER:www-data storage bootstrap cache
```

### Email Tidak Terkirim

- Cek kredensial Mailtrap di `.env`
- Cek `storage/logs/laravel.log` untuk error
- Pastikan firewall tidak block port 2525

### Port 8000 Sudah Dipakai

Ganti port:

```bash
php artisan serve --port=8080
```

Dan update `APP_URL` di `.env` ke `http://127.0.0.1:8080`.

## 🛠️ Perintah Artisan Penting

```bash
# Development
php artisan serve                       # Start dev server
php artisan migrate                     # Run migrations
php artisan db:seed                     # Run seeders
php artisan tinker                      # Interactive shell

# Cache
php artisan cache:clear
php artisan config:clear
php artisan view:clear
php artisan route:clear
php artisan optimize:clear               # Clear all caches

# Review System
php artisan reviews:send-daily-invitations  # Trigger review emails (scheduled daily)

# Queue (kalau pakai QUEUE_CONNECTION=database)
php artisan queue:work
php artisan queue:listen                 # Auto-reload saat develop

# Routes
php artisan route:list                  # List semua route
```
